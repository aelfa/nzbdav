# Changelog

Release history is maintained automatically from conventional commits:

- **[CHANGELOG.md on GitHub](https://github.com/nzbdav/nzbdav/blob/main/CHANGELOG.md)**
- **[GitHub Releases](https://github.com/nzbdav/nzbdav/releases)** — downloadable tags and notes
